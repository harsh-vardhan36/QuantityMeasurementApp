# QuantityMeasurementApp

## UC1 - Feet Equality
> This branch contains the code demonstrating the Feet class. It essentially illustrates how a single unit compares to itself.  A unit must be compared to itself first before being compared to others.
---
### Date : 17 Feb 2026
* I worked on feature 1, defining a single unit of measurement: feet.
> * I developed a Feet class and performed the following operations:
>1. Override equals() from the Object class
>2. Check if the object is the same reference (this == obj)
>3. Check if the object is null or a different type
>4. Cast to Feet type safely
>5. Compare double values using Double.compare() instead of == operator
